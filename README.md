ViralClip Website (scaffold)

This repository contains a minimal static scaffold for the ViralClip marketing site.

Files:
- index.html: Home page
- pricing.html: Pricing details
- beta.html: Beta sign-up
- assets/css/style.css: Basic styles
- assets/js/main.js: Minimal JS

To view the site, open `index.html` in your browser (no server required for basic testing).
